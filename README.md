# PureFeed

**PureFeed** 是一款 Chrome 浏览器插件，旨在打造纯净的时间线体验，让你专注于高价值内容。它可以帮助你隐藏短文本推文，按媒体类型过滤，并控制你看到的内容语言。

## 功能特性

### 核心功能
- **双模式切换**：原始模式（无过滤）和精简模式（应用过滤规则）
- **短文本过滤**：隐藏少于 45 个中文字符或 30 个英文单词的推文
- **媒体类型过滤**：
  - 只显示带图片的推文
  - 只显示带视频的推文
  - 只显示同时包含图片和视频的推文
- **语言过滤**：支持仅中文、仅英文或全部语言
- **关键词过滤**：隐藏包含特定关键词的推文（黑名单）
- **互动量过滤**：根据浏览量、点赞数、转发数、收藏数设置最低阈值
- **账号白名单**：白名单账号的推文始终显示，不受其他过滤规则影响
- **实时更新**：设置更改后立即应用，无需刷新页面
- **状态指示**：浏览器工具栏图标显示当前模式状态

## 🛠 安装说明

由于这是一个 MVP 版本的插件，目前需要通过“开发者模式”安装：

1.  **克隆或下载** 此仓库到你的本地机器。
2.  打开 Google Chrome 并访问 `chrome://extensions`。
3.  在右上角开启 **开发者模式 (Developer mode)**。
4.  点击 **加载已解压的扩展程序 (Load unpacked)**。
5.  选择 `x-cleaner` 目录。

## 📖 使用指南

1.  点击 Chrome 工具栏中的 **X Cleaner** 图标。
2.  **模式选择**：
    - **原始模式**：不应用任何过滤。
    - **精简模式**：应用你选择的过滤规则。
3.  **配置过滤规则**：
    - 开启“隐藏短文本”开关以去除噪音。
    - 勾选“只显示...”复选框以专注于媒体内容。
    - 根据需要选择语言偏好。
4.  **白名单**：
    - 在文本区域输入用户名（用逗号分隔），确保不错过他们的更新。
5.  点击 **保存设置** 以立即应用更改。

## 🔧 技术细节

- **Manifest V3**：符合最新的 Chrome 扩展标准。
- **隐私保护**：完全在本地运行。没有任何数据发送到外部服务器。
- **高性能**：使用 `MutationObserver` 高效处理滚动时的新推文，不会拖慢页面速度。
